# Hibachi2Party

Static website for [hibachi2partys.com](https://hibachi2partys.com) — mobile hibachi catering at home.

## Deploy (Cloudflare Pages)

Connect this repo to Cloudflare Pages with:

- **Build command:** (none)
- **Build output directory:** `/` (repository root)

Or deploy with Wrangler:

```bash
npx wrangler deploy
```

## Contents

Pre-built HTML, CSS, JS, `_redirects`, and `sitemap.xml`. No build step required.
